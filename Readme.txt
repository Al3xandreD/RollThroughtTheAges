									Bienvenue dans le jeu Roll Through The Ages fait par nos soins !



Tout d'abord, nous allons commencer par décrire les différents fichiers qui composent ce dossier:

CodeavecIHM: c'est celui qu'il faut ouvrir pour jouer à notre jeu avec l'IHM ( avec des magnifiques illustrations et un son légendaire)

Codesansihm: c'est le code que nous avions rendu lors de la première partie. Il fonctionne avec la console.

Bdd: c'est le fichier qui construit notre base de donnée

Unitest : c'est le fichier qui effectue différents test pour verifier notre programme

Diffentes images : Ce sont les illustrions et le plateau de jeu, nous espérons que tout s'ouvrira correctement.



Certains module seront nécessaires pour profiter pleinement:

-prettytable (creation tableau sur un fichier compte rendu)

-pygame (pour mettre de la musique)

-Pyqt5( pour l'Ihm)


Premier pas pour lancer CodeavecIHM. Vous possédez différentes options lors du lancement du jeu :

-Visualisation des règles ( nous avons mis la version pour Windows, si vous êtes sur MacOs, vous pouvez changer ligne 411)

-Quitter (Vous partez déjà ?)

-Nouvelle Partie 

-Continuer et le score board sont en développement pour la version beta


Nouvelle Partie:

Vous devrez rentrer vos pseudos à tour de rôle pour vous inscrire dans la partie. Si vous avez déjà joué, vous possédez un code ( Game_id) pour continuer à alimenter votre profil.
Si vous n'avez pas de code, le jeu vous en fournira un pour les prochaines parties.



Ensuite vous n'avez qu'à jouer !


Merci de votre lecture et bonne Partie !
